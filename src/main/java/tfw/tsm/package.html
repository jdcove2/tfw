This is a transaction based state management component framework. It uses a publish and subscribe communication infrastructure. Components and communcations channels are organized hierarchically in a tree structure. Define rooted. Build components from the leaves to the components. Lazy evaluation of component structures.

The framework supports two kinds of transactions. Those that are initiated by a change of state and those that are initiated by structural changes.

There are two different kinds of state changes in the framework. Those that initiate a new transaction and those that occur within a transaction.
